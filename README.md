# Floppy Dude

A Flappy Bird-style game built in C# using Windows Forms, inspired by Jacksfilms' parody.

## Tech Stack
- C#
- Windows Forms (.NET)
- Visual Studio

## Features
- Real-time game loop using Windows Forms Timer
- Gravity simulation and Space bar controls
- Collision detection with obstacles, ground, and ceiling
- Dynamic difficulty — obstacle speed increases after score 5
- Live score tracking
- Restart button on game over

## How to Play
- Press **Space** to make Floppy Dude fly up
- Avoid the brick obstacles, ground, and ceiling
- Every obstacle passed increases your score
- Game gets faster after score 5
- Press **Restart** when game over to play again

## Setup
1. Clone the repository
2. Open `Floppy Dude.sln` in Visual Studio
3. Build and run the project
